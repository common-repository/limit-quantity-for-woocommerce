This plugin adds input box in the WooCommerce product backend allowing to choose a maximum quantity allowed on per product basis.

== Description ==
This plugin adds input box in the WooCommerce product backend allowing to choose a maximum quantity allowed on per product basis.

== Installation ==


    Upload Limit quantity for WooCommerce to the /wp-content/plugins/ directory

    Activate the plugin through the ‘Plugins’ menu in WordPress
